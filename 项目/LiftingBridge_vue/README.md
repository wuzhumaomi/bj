# projectvue

## 项目安装依赖
```
npm install
```

### 启动项目--开发模式
```
npm run serve
```

### 打包项目--生产模式
```
npm run build
```

### 整理和修复文件
```
npm run lint
```

### 自定义配置帮助文档
See [Configuration Reference](https://cli.vuejs.org/config/).

# 开发须知

## 接口定义
```
如果需要传递json数据到后台，定义接口时需要在最后加上“-json”
```

##注意事项

###1、二次确认弹出层
```
除添加外的所有操作都应加上二次确认弹出层
```

###2、操作后提示信息
```
成功统一显示“操作成功”提示
失败统一显示“操作失败”提示
```

###3、页面上所有“优先级”显示样式规则
```
“加急” -- 字体加红加粗，“紧急” -- 字体加红，“一般” -- 字体不变
```
